初窥Vue.js

Vue.js（读音 /vjuː/, 类似于 view） 是一套构建用户界面的 渐进式框架。与其他重量级框架不同的是，Vue 采用自底向上增量开发的设计。Vue 的核心库只关注视图层，并且非常容易学习，非常容易与其它库或已有项目整合。另一方面，Vue 完全有能力驱动采用单文件组件和 Vue 生态系统支持的库开发的复杂单页应用。 Vue.js 的目标是通过尽可能简单的 API 实现响应的数据绑定和组合的视图组件。

Vue.js 的核心是一个允许你采用简洁的模板语法来声明式的将数据渲染进 DOM 的系统

它具有独有的声明式渲染！

            <div id='app'>
                 {{message}}
            </div>
            var app = new Vue({
                 el:'#app',
                 data:{
                 message:'hello Vue!'
                 }
            })

vue 使用了几个很方便的指令 大大加快了开发效率

条件指令v-if：

            <div id="app-3">
            <p v-if="seen">Now you see me</p>
            </div>

            var app3 = new Vue({
                    el: '#app-3',
                    data: {
                      seen: true
                    }
            })

循环指令 v-for

            <div id="app-4">
                <ol>
                  <li v-for="todo in todos">
                    {{ todo.text }}
                  </li>
                </ol>
            </div>

处理用户输入：v-model 使数据双向绑定

            <div id="app-6">
                  <p>{{ message }}</p>
                  <input v-model="message">
            </div>

            var app6 = new Vue({
                  el: '#app-6',
                  data: {
                  message: 'Hello Vue!'
                  }
            })

绑定vue事件方法：v-on

            使用.prevent修饰符告诉v-on指令对于触发事件调用event.preventDefault();
            <a v-on:click = 'doSomething'></a>
             也可以简写：
            <a @click = 'doSomething'></a>
